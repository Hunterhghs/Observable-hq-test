Integration Methods

Simplest approach — works in any HTML page, WordPress, Notion, CMS. Embed the full page as an iframe.

<!-- Embed any Observable page in your website --> <iframe src="https://hheuristics.com/dashboards/nc-atlas" style="width:100%; height:600px; border:none;" loading="lazy" ></iframe> <!-- Pro: add API key for private notebooks --> <iframe src="https://api.observablehq.com/d/abc123@latest?v=3&api_key=YOUR_KEY" ></iframe> <!-- Pro: remove attribution banner automatically -->

For deeper control — embed individual cells as reactive JavaScript modules. Style with your own CSS. Cells update live.

// Import the Observable Runtime import { Runtime, Inspector } from "@observablehq/runtime"; // Import your notebook as a module import notebook from "@hheuristics/nc-economic-atlas"; // Render specific cells into your DOM const runtime = new Runtime(); const main = runtime.module(notebook, name => { if (name === "countyMap") { return new Inspector(document.querySelector("#my-map")); } if (name === "yearSlider") { return new Inspector(document.querySelector("#my-slider")); } }); // Cells are reactive — changing yearSlider updates countyMap automatically

For React/Next.js apps — import cells as React components using @observablehq/runtime's React bindings.

// Install: npm install @observablehq/runtime import { Runtime, Inspector } from "@observablehq/runtime"; import notebook from "@hheuristics/nc-economic-atlas"; function DashboardEmbed() { const mapRef = useRef(null); useEffect(() => { const runtime = new Runtime(); const main = runtime.module(notebook, name => { if (name === "visualization") return new Inspector(mapRef.current); }); return () => runtime.dispose(); }, []); return <div ref={mapRef} />; }

Build the entire site with Observable Framework and deploy as a static site to GitHub Pages, Cloudflare, or any host.

# Build and deploy your Framework site npm run build # → dist/ folder with static HTML, JS, CSS # Deploy to GitHub Pages git push origin main # GitHub Actions auto-builds and deploys # Or deploy anywhere: npx wrangler pages deploy dist # Cloudflare Pages aws s3 sync dist/ s3://my-bucket # AWS S3 + CloudFront netlify deploy --dir=dist # Netlify
